Simple Database Manager for backup and restore.

== Description ==

Using DB Manager is so easy.

It allows to download, restore, and delete backups of database.


== Installation ==

Just install from your WordPress "Plugins > Add New" screen and all will be done automatically. Manual installation:

1. Upload the zip file and unzip it in the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to `Settings > DB Manager` and enable the options that you need.
